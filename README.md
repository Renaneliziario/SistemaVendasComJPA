# Sistema de Vendas com JPA

Projeto de estudo desenvolvido durante um curso de Java, com foco em **JPA** e mapeamento objeto-relacional.  
O objetivo é simular um pequeno sistema de vendas, praticando entidades, relacionamentos e operações de persistência.

## Tecnologias utilizadas

- Java
- JPA / Hibernate
- Maven
- Eclipse (projeto originalmente criado no Eclipse)

## Objetivos do projeto

- Praticar criação de entidades e relacionamentos (OneToMany, ManyToOne, etc.).
- Exercitar o uso de `EntityManager` para CRUD.
- Entender como configurar JPA em um projeto Maven.
